# ResumeGen Pro - 专业简历生成器

一个现代化、功能丰富的在线简历生成器，帮助您在几分钟内创建专业简历。

🔗 **在线体验**: [https://kanshaowz.github.io/resumegen](https://kanshaowz.github.io/resumegen)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-2.0-green.svg)

## ✨ 核心特性

### 🎨 8+ 精美模板
- **现代专业** - 简洁现代的布局，适合各行各业
- **经典优雅** - 传统正式的设计，适合保守行业
- **极简风格** - 清爽简约，突出内容本身
- **创意侧边栏** - 独特的侧边栏设计，视觉突出
- **双栏布局** - 高效利用空间，信息密度高
- **高管风格** - 大气稳重，适合管理岗位
- **紧凑实用** - 一页纸设计，信息集中
- **科技风格** - 现代科技感，适合IT行业

### 📋 分步表单流程
1. **个人信息** - 填写基本信息和职业概述
2. **工作经历** - 添加工作经验，支持多段经历
3. **教育背景** - 记录学历和教育经历
4. **技能专长** - 展示专业技能和语言能力

### 🤖 AI 智能助手
- **个人简介优化** - AI 帮您生成专业的个人简介
- **技能推荐** - 根据职位智能推荐相关技能
- **经历描述优化** - 优化工作描述，突出成就

### 🎨 8种配色主题
海军蓝、森林绿、勃艮第红、石板灰、靛蓝、玫瑰红、翡翠绿、琥珀金

### 📤 多格式导出
- **PDF 导出** - 专业打印质量，推荐格式
- **Word 导出** - 可编辑格式，方便后续修改
- **直接打印** - 一键打印简历

### 📱 响应式设计
- 完美支持桌面端和移动端
- 实时预览与编辑同步
- 数据本地存储，自动保存

## 🚀 快速开始

### 在线使用
直接访问 [https://kanshaowz.github.io/resumegen](https://kanshaowz.github.io/resumegen) 即可使用

### 本地运行
```bash
git clone https://github.com/kanshaowz/resumegen.git
cd resumegen
# 使用任意本地服务器打开，例如：
python -m http.server 8000
# 或
npx serve
```

## 🛠️ 技术栈

- **前端**: 纯 HTML5 + CSS3 + JavaScript (ES6+)
- **样式**: CSS Variables, Flexbox, Grid
- **PDF生成**: html2pdf.js
- **图标**: Font Awesome
- **字体**: Inter + Noto Sans SC

resumegen/

## 📝 使用指南

1. **选择模板** - 从8种专业模板中选择您喜欢的样式
2. **填写信息** - 按步骤填写个人信息、工作经历、教育背景和技能
3. **使用AI助手** - 点击 AI 按钮获取智能建议
4. **调整主题** - 选择喜欢的配色方案
5. **预览效果** - 实时查看简历效果
6. **导出简历** - 导出为 PDF 或 Word 格式
