# ResumeGen - 简历生成器

🎨 **现代简约的简历生成工具**，支持实时预览和PDF下载。

![ResumeGen Screenshot](./screenshot.png)

## ✨ 功能特点

- 🎨 **深蓝色商务风格** - 专业、现代的视觉设计
- 📝 **表单填写** - 直观的信息录入界面
- 👁️ **实时预览** - 即时查看简历效果
- 📄 **PDF下载** - 一键导出高质量PDF
- 🌍 **5语言支持** - 英文、中文、西班牙语、日语、韩语
- 💾 **本地存储** - 自动保存数据，刷新不丢失

## 📝 使用说明

1. **填写个人信息** - 在左侧表单中输入基本信息
2. **添加工作经历** - 点击"添加"按钮增加工作经历
3. **添加教育背景** - 录入学历信息
4. **填写技能** - 用逗号分隔多个技能
5. **实时预览** - 右侧实时显示简历效果
6. **下载PDF** - 点击下载按钮保存简历

## 🎨 设计特色

- **深蓝配色方案** - #0a1628 到 #1e3a5f 的渐变
- **毛玻璃效果** - backdrop-filter 实现的现代UI
- **金色点缀** - 下载按钮使用金色强调
- **响应式布局** - 适配桌面和移动设备
